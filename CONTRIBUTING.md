# Contributing to Smart Context Plugin

Thank you for your interest in contributing to the Smart Context Plugin! This document outlines the process for contributing to this project.

## How to Contribute

### Reporting Issues
- Use the GitHub issue tracker to report bugs or request features
- Include detailed steps to reproduce the issue
- Provide information about your environment (Node.js version, Claude Code version, etc.)

### Submitting Pull Requests
1. Fork the repository
2. Create a feature branch
3. Make your changes
4. Write tests for your changes
5. Ensure all tests pass
6. Submit a pull request

### Code Style
- Follow the existing code style
- Write clear, descriptive commit messages
- Document your changes
- Ensure code is properly formatted

## Development Setup

### Prerequisites
- Node.js 14+
- Claude Code 2.0+

### Installation
```bash
git clone https://github.com/your-username/smart-context-plugin.git
cd smart-context-plugin
npm install
```

### Running Tests
```bash
npm test
```

### Building
```bash
npm run build
```
